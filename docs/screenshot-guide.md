# JobMate AI 产品截图指南

## 服务地址
http://localhost:8080

---

## 截图位置建议

### 1. 登录页面（Login）
**路径**：http://localhost:8080/#/login

**截图要点**：
- 渐变紫色背景
- Logo + 产品名称
- 登录/注册 Tab 切换
- 表单输入框

**用途**：PPT 封面或产品介绍页

---

### 2. 首页（Home）
**路径**：http://localhost:8080/#/

**截图要点**：
- 整体布局展示
- 左侧导航菜单
- 右上角用户信息/登录按钮
- 欢迎语和功能入口

**用途**：产品概览

---

### 3. 简历优化页面（ResumeOptimize）
**路径**：http://localhost:8080/#/resume

**需要2张截图**：

#### 截图 A：输入界面
- 目标职位输入框
- 简历内容文本框
- 文件上传区域
- "开始 AI 分析" 按钮

#### 截图 B：分析结果
**操作步骤**：
1. 在文本框粘贴以下简历内容：

```
个人信息
姓名：张三
电话：13800138000
邮箱：zhangsan@email.com

教育背景
2020-2024 北京大学 计算机科学与技术 本科

工作经历
2024.07-至今 某科技公司 后端开发工程师
- 负责公司核心业务系统开发
- 使用 Java、Spring Boot、MySQL
- 参与过多个项目

项目经历
电商平台后端开发
- 使用 Spring Boot + MySQL + Redis
- 实现了商品管理、订单管理功能
- 提高了系统性能

技能
Java、Spring Boot、MySQL、Redis、Git
```

2. 点击"开始 AI 分析"（需要配置 API）
3. 截取分析结果页面：
   - 评分圆环（如：78分）
   - 问题诊断列表
   - 改进建议列表

**用途**：PPT 核心功能演示

---

### 4. JD 分析页面（JDAnalyze）
**路径**：http://localhost:8080/#/jd

**截图要点**：
- JD 粘贴区域
- 简历摘要输入（可选）
- 匹配度分析结果
- 技能对比（已具备 vs 待提升）

**示例 JD 文本**：
```
后端开发工程师

岗位职责：
1. 负责公司核心业务系统的设计与开发
2. 参与系统架构设计和技术选型
3. 优化系统性能，解决技术难点

任职要求：
1. 本科及以上学历，计算机相关专业
2. 3年以上 Java 开发经验
3. 精通 Spring Boot、MySQL、Redis
4. 熟悉微服务架构，有分布式系统开发经验
5. 熟悉 Linux 操作系统
6. 良好的沟通能力和团队协作精神

薪资范围：20k-35k
```

**用途**：PPT 核心功能演示

---

### 5. AI 设置页面（Settings）
**路径**：http://localhost:8080/#/settings

**截图要点**：
- AI 提供商选择下拉框
- API Key 输入框
- 模型选择
- "保存并测试" 按钮
- 配置说明卡片

**用途**：展示多提供商支持和隐私安全

---

### 6. 面试准备页面（InterviewPrep）
**路径**：http://localhost:8080/#/interview

**截图要点**：
- 公司名称输入（如：字节跳动）
- 目标职位输入（如：后端开发工程师）
- 生成的情报报告：
  - 公司概况
  - 企业文化
  - 技术栈
  - 预测面试题列表

**用途**：PPT 核心功能演示

---

### 7. 登录后的界面
**操作步骤**：
1. 点击右上角"登录"按钮
2. 注册一个新账号（用户名、邮箱、密码）
3. 登录成功后截图：
   - 右上角显示用户名和头像
   - 下拉菜单展示"个人中心"和"退出登录"

**用途**：展示用户系统功能

## 截图工具推荐

| 工具 | 特点 | 操作 |
|------|------|------|
| **Windows 自带** | 无需安装 | Win + Shift + S |
| **Snipaste** | 可贴图、标注 | F1 截图，F3 贴图 |
| **ShareX** | 功能强大 | PrintScreen |
| **浏览器开发者工具** | 全页面截图 | F12 → Ctrl+Shift+P → "full" |

## PPT 截图替换说明

1. 打开 PPT：`JobMate_AI_参赛PPT_优化版.pptx`
2. 找到第 8 页"产品演示"
3. 右键点击截图占位框
4. 选择"更改图片" → "来自文件"
5. 选择你的截图替换

## 截图规格建议

- **分辨率**：1920x1080 或更高
- **格式**：PNG（无损）
- **比例**：16:9 或 4:3
- **重点突出**：使用红色方框标注关键区域
