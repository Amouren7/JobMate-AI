## 1. 产品背景与痛点

### 求职者面临的三大困境

1. **简历不会写**
   - 不知道 HR 想看什么
   - 项目描述太空，缺乏量化
   - 无法通过 ATS（applicant tracking system）筛选

2. **投递没反馈**
   - 海投效率低，不知道哪些岗位适合自己
   - 不了解岗位真实要求
   - 简历与 JD 匹配度低

3. **面试准备难**
   - 不了解目标公司业务
   - 不知道面试官会问什么
   - 缺乏系统性准备方法

## 3. 核心功能

### 3.1 简历智能优化中心

**功能描述**：
- ✅ 支持文本简历输入
- ✅ AI 深度解析简历结构（技能、经历、项目）
- ✅ 生成匹配度评分（0-100分）
- ✅ 识别简历问题并给出改进建议
- ✅ AI 一键生成优化后的简历内容

**技术亮点**：
- 使用真实 LLM 进行深度分析
- 基于目标职位进行针对性优化
- 自动添加量化指标建议
- 智能提取行业关键词

**使用方式**：
1. 进入「简历优化」页面
2. （可选）输入目标职位
3. 粘贴简历内容
4. 点击「开始 AI 分析」
5. 查看 AI 生成的评分和建议
6. 点击「AI 一键优化」生成改进版本

---

### 3.2 JD 智能分析

**功能描述**：
- ✅ 粘贴岗位 JD 描述，AI 自动解析
- ✅ 提取核心技能要求
- ✅ 分析简历与 JD 的匹配度（0-100%）
- ✅ 识别已具备技能 vs 待提升技能
- ✅ 生成针对性投递建议

**分析维度**：
- 硬性技能匹配度
- 软性技能要求
- JD 高频关键词
- 竞争力评估
- 薪资范围匹配（如有）

**使用方式**：
1. 进入「JD分析」页面
2. 粘贴职位描述（JD）
3. （可选）输入简历摘要用于匹配
4. 点击「开始 AI 分析」
5. 查看匹配度评分和建议

---

### 3.3 面试准备工坊

**功能描述**：
- ✅ 输入公司名称，AI 生成情报报告
- ✅ 公司背景、业务、技术栈分析
- ✅ 企业文化特点总结
- ✅ AI 预测面试题（技术题、行为题、情景题）
- ✅ 提供参考回答思路

**情报报告内容**：
- 公司概况（行业、规模、发展阶段）

**使用方式**：

4. 点击「生成 AI 情报报告」

### 7.2 服务器部署

**使用 Nginx**：

```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /path/to/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

**使用 Node.js**:

```bash
cd dist
npx serve -l 8080
```

## 9. 常见问题

### Q1: 为什么需要自行配置 API Key？

A: 
- 前端直接调用 AI API，无需后端服务
- 个人用户可独立使用
- API Key 仅本地存储，安全可靠
- 用户自主选择 AI 提供商和模型

### Q2: 连接失败怎么办？

A:
- **CORS 错误**：使用中转服务（OneAPI）
- **Key 无效**：检查 Key 是否正确，是否已激活
- **网络问题**：确认域名可访问
- **模型不存在**：检查模型名称拼写

### Q3: 如何降低使用成本？

A:
- 使用阿里云 DashScope（国内便宜）
- 选择轻量级模型（如 qwen-turbo）
- 减少单次分析的简历/JD 长度

### Q4: 支持哪些文件格式？

A:
- 当前版本支持文本直接粘贴
- 文件上传功能需配合后端实现
